h1. Getting started


Download the appropriate client binary for your platform from "downloads page":/jark/downloads.html
Untar and copy the jark-0.4-`platform` script to PATH/jark

To install the server:
<pre class="terminal"><code>server> jark install
</code></pre>

Ensure that JAVA_HOME is set appropriately.

The install command extracts relevant files to ~/.cljr and ~/.config/jark

The VM can be started on any available port. 

<pre class="terminal"><code>server> jark vm start [--port]</code></pre>

To connect, do
<pre class="terminal"><code>client> jark vm connect [--host] [--port]
</code></pre>

In the above, host defaults to localhost and port to 9000. Any number of JVM instances can be started.

Now, just launch a repl and write some clojure!
<pre class="terminal"><code>client> jark repl</code></pre>or simply
</code></pre>

To get the status of the current connection:
<pre class="terminal"><code>client> jark self status</code></pre>

Jark commands can be run from within the repl, using the '/' prefix:
<pre class="terminal"><code>user>> /cp list
....
</code></pre>

We can switch on/off the nrepl debug-mode
<pre class="terminal"><code>user>> /debug on
debug is ON
user>> (+ 2 2)
put : (+ 2 2)
got : {'status': 'done', 'ns': 'user', 'id': 'localhost:9000-repl', 'value': '4\n'}
4
user>> /debug off
</code></pre>
