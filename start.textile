h2. Getting started

Download the jark self extracting script "server and client":https://github.com/downloads/icylisper/jark/jark-0.3 or only the "client":https://github.com/downloads/icylisper/jark/jark-0.3-client

<pre class="terminal"><code>server> cp jark-0.3 ~/bin/jark (PATH)
server> jark self install
</code></pre>
or to install just the client
<pre class="terminal"><code>client> jark-0.3-client ~/bin/jark (PATH)
client> jark self install</code></pre>

The install command extracts relevant files to ~/.cljr and ~/.config/jark

We can start the VM on any available port. 

<pre class="terminal"><code>server> jark vm start [--port]</code></pre>

We can connect from any remote host:
<pre class="terminal"><code>client> jark vm connect [--host] [--port]
</code></pre>

In the above, host defaults to localhost and port to 2113. Any number of JVM instances can be started.

Now, just launch a repl and write some clojure!
<pre class="terminal"><code>client> jark ns repl NAMESPACE</code></pre>or simply
<pre class="terminal"><code>client> jark repl [NAMESPACE]
</code></pre>

Have a look at the other sections for some useful utilities.
